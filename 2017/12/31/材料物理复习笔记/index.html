关于 Read the Docs 的碎碎念

  • 在 Read the Docs 新建项目的时候，项目名会被直接用做网址，因此最好全英文（汉字会被直接忽略）。之后可以在“管理”选项中再把项目名改中文。
  • 选择 Cython2 还是 Cython3 构建似乎没什么区别。
  • 隐私级别选择“保护”似乎也没什么用，构建之后还是公开级别的。
